Program Crypto

Program displays various information related to cryptocurrencies. 
It implies the use open APIs:
CryptingUp: https://www.cryptingup.com/apidoc/

Functionality:
It is a multi-page application with navigation.
The main page displays currencies, exchanges and markets.
View detailed information about the currency: price, volume, price change ets
Possibility of searching for currency by name or code.
Convert one currency to another
